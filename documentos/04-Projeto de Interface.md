
# Projeto de Interface
## User Flow

O User Flow, ou fluxo do usuário, é a representação visual das etapas que um usuário segue ao interagir com um produto ou serviço. Ele descreve o caminho desde o início até a conclusão de uma tarefa específica dentro do sistema.

O User Flow é uma ferramenta essencial para entender e comunicar o funcionamento da Aplicação Web de forma clara e concisa. Ele mapeia as diferentes etapas e caminhos que um usuário pode seguir, desde o acesso inicial até a conclusão de uma tarefa ou objetivo. Isso inclui pontos de decisão, desvios possíveis e retornos ao longo do processo, como representado a seguir (**Figura 9**).

**Figura 9 - User Flow**

No User Flow do Projeto VAT, os usuários possuem diferentes opções de interação. Eles poderão explorar os projetos sociais em destaque na Página Inicial/Principal e realizar buscas por ONG's e projetos específicos, utilizando a função de pesquisa. Além disso, os usuários poderão fazer doações diretamente pelo site. Após explorar as iniciativas socais, há a opção de se cadastrarem como Voluntários/Doadores e/ou cadastrar uma Instituição Social. Se já forem cadastrados e perderem a senha de cadastro, também será possivel recupera-lá. Os usuários também poderão acessar o seu Perfil e as Configuções. Uma funcionalidade adicional é o Chat Integrado, que permitirá que os usuários entrem em contato não só com as ONG's, mas também com outros voluntários. Esse recurso estará disponível para comunicação com a equipe de suporte do site, assim como com as Instituições ou voluntários com quem desejam colaborar.

## Protótipo

O Projeto foi desenvolvido com base nos requisitos funcionais e não funcionais estabelecidos, visando uma experiência do usuário fluida e intuitiva. Para isso, foram adotadas uma identidade visual padronizada entre as telas, a fim de proporcionar um ambiente familiar e confortável aos usuários.

Considerando a natureza do sistema, que depende da participação ativa do usuário para alcançar seus objetivos, foi determinada a necessidade de criação de cadastros/perfis de usuário. Essa medida visa facilitar a correlação de títulos com preferências e personalizações, garantindo que os resultados desejados sejam alcançados de forma eficaz.

Além disso, foram desenvolvidos protótipos de baixa fidelidade para representar de forma simplificada o design da interface e o relacionamento entre suas páginas. Esses protótipos serão utilizados para validar os requisitos e realizar ajustes, minimizando o impacto na codificação da aplicação.

As telas do sistema foram projetadas com uma navegação clara e simples, priorizando a acessibilidade e a eficiência de uso. Elementos de interface, como botões e ícones, foram estrategicamente posicionados para otimizar a interatividade. A organização visual e a hierarquia de informações foram cuidadosamente planejadas para garantir que os usuários encontrem facilmente o que procuram. Com essas medidas, espera-se que sirva como base sólida para o desenvolvimento subsequente de um protótipo de alta fidelidade que, supere as expectativas dos usuários e promova uma conexão mais forte e duradoura com o público-alvo.

## Protótipo de baixa fidelidade

### HomePage ou Página Principal

A tela da Página Principal exibe todo o conteúdo acessível do site. É a primeira tela que os usuários visualizarão ao acessarem o site, onde poderão realizar buscas, encontrarão informações sobre as ONG's e os relatos dos voluntários. No canto superior esquerdo, estará localizada a logomarca, enquanto no canto superior direito, encontra-se a opção de Início, Mensagens, Contato e Login.

### Tela de Login

A tela de Login nos apresentará o local onde o usuário poderá efetuar o seu login, por meio do preenchimento do Email e Senha, para ter acesso ao sistema. Caso seja o primeiro acesso, o usuário deverá realizar o cadastro por meio do link "Cadastre-se". Caso o usuário precise redefinir sua senha ou recupera-lá, ele poderá acessar o link “Esqueceu sua senha?”.

### Tela de Doação 
Ao clicar em "Doe agora", os usuários serão redirecionados para esta tela. A tela de Doação será um elemento crucial que permitirá aos usuários contribuírem financeiramente para apoiar as causas e projetos da organização. Nela, o usuário poderá informar o valor que deseja doar e para qual Instituição Social. Após a confirmação, o mesmo será redirecionado para a tela do Pagamento.
